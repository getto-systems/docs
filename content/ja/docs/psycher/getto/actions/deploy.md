  デプロイ完了の通知

デプロイ完了の通知を行う。

- success : 「成功」のリアクションを追加する
- failure : 「失敗」のリアクションを追加する


## success

以下の条件を全て満たすなら、処理を行う。

- 結果が「成功」にマッチする

処理の内容は以下の通り。

- 成功のリアクション Message をつける


## failure

以下の条件を全て満たすなら、処理を行う。

- 結果が「失敗」にマッチする

処理の内容は以下の通り。

- 失敗のリアクション Message をつける
