{{< /blocks/cover >}} {{% blocks/lead %}} げっとシステムが集める情報について {{% /blocks/lead %}} {{< blocks/section color="info" >}} {{% blocks/feature icon="fa-shoe-prints" title="アクセス履歴" %}} げっとシステムが提供するサービス内のアクセス履歴を記録します {{% /blocks/feature %}} {{% blocks/feature icon="fa-envelope" title="メールアドレス" %}} ユーザー登録時にメールアドレスを要求するサービスがあります {{% /blocks/feature %}} {{% blocks/feature icon="fa-key" title="ログインID / パスワード" %}} サービスを利用するためにログインID とパスワードを登録する必要があります {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="primary" >}}

げっとシステムが情報を集める理由

{{< /blocks/section >}} {{< blocks/section color="info" >}} {{% blocks/feature icon="fa-shoe-prints" title="アクセス履歴" %}} どの機能にアクセスし、どんな操作を実行したかを解析し、サービスの改善を行います 現在のところ、サービス内のアクセス履歴を削除する機能はありません {{% /blocks/feature %}} {{% blocks/feature icon="fa-envelope" title="メールアドレス" %}} パスワードがわからなくなった場合に一時的なトークンを送信します メールアドレスはサービスの設定ページから削除できます {{% /blocks/feature %}} {{% blocks/feature icon="fa-key" title="ログインID / パスワード" %}} パスワードは暗号化されて保管されます サービスへのログイン以外でこの情報を利用することはありません {{% /blocks/feature %}} {{< /blocks/section >}}
